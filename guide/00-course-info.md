# 实验须知

**课程名称：** 计算机设计与实践（2026夏季）

**开课单位：** 哈尔滨工业大学（深圳）

---

## 课程概述

本课程为 2 人小组合作项目课程，最终目标是实现一个流水线 SoC（System on Chip），能够在 FPGA 板上运行 CoreMark 或 LLAMA2 推理程序。

课程基于 **miniRV** 指令集（RV32I 的子集）进行 CPU 设计，使用 **EGO1 FPGA 开发板**作为硬件验证平台。

---

## 小组安排

- 2 人一组，可自行组队或由任课教师分配
- 小组成员需要分工协作，共同完成实验任务

---

## 实验内容

### 实验一：单周期 CPU

设计并实现支持 **miniRV** 或 **miniLA** 指令集的单周期 CPU。

- 完成数据通路设计、控制单元设计、功能部件设计
- 实现所有指令的硬件电路
- 通过 **Basic Trace** 测试

#### 小组分工（参考）

| 成员 | 任务 |
|------|------|
| 成员 1 | 完成 A 组指令的设计与实现 |
| 成员 2 | 完成 B 组指令的设计与实现 |
| 协作 | 整合两组指令，完成完整单周期 CPU 的调试与验证 |

### 实验二：流水线 CPU 与 SoC

将单周期 CPU 改造为流水线 CPU，并添加以下组件：

- 指令缓存（ICache）
- 数据缓存（DCache）
- AXI 总线接口

最终在 FPGA 板上运行 CoreMark 或 LLAMA2 推理程序，通过 **AXI Trace** 测试。

## 检查与考核

### 现场检查项目

1. **单周期 CPU 数据通路图** —— 完整的数据通路图
2. **SoC 运行展示** —— 在 FPGA 上运行 CoreMark 或 LLAMA2 程序

### Trace 测试

- **Basic Trace** 和 **AXI Trace** 由 CI 自动评测，无需课堂现场检查
- 所有 Trace 测试均可多次提交，以最后一次结果为准
