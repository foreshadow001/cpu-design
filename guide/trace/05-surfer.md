# D. Surfer 使用指南

[Surfer](https://surfer-project.org/) 是一个开源、可扩展、跨平台的数字电路波形查看工具。

波形文件和配置文件都可以直接拖拽打开，如下图所示。

## 2. 添加和删除信号

首先在左上方的 Scopes 区域找到想要查看信号的模块，单击选中；然后在右侧选择想要把信号添加到哪个信号下面；最后再点击 Variables 区域下的信号，如下图所示。

> [图片描述] Surfer 界面标注：左侧 Scopes 区域、中间 Variables 区域、右侧已添加信号区域

只要鼠标点击 Variables 内的信号，这些信号就会被添加到右侧被选中的信号下方。比如在上图中，如果点击 Variables 内的 `is_mul_div` 信号，则该信号就会被添加到右侧的 `ifetch_inst` 信号的下方。

建议先在右侧选好希望添加信号的位置，再点击 Variables 内的信号，否则容易使得波形中的信号杂乱无章，不利于分析和调试。

添加信号时，还可以在 "Variables" 字样右侧的 "Filter" 搜索框内输入信号名称进行搜索。

删除信号时，只需点击右侧的某个信号，按下 `Del` 键即可。

## 3. 波形缩放

Surfer 支持两种波形缩放方式：

第一种是按住 `Ctrl` 键，再滚动鼠标滚轮。此时，Surfer 将以鼠标指针指向的竖直线为中心，缩放波形。

第二种方法是点击工具栏的放大、缩小按钮，如下图所示。

> [图片描述] Surfer 工具栏的放大、缩小按钮位置

## 4. 波形定位

Surfer 支持滚动鼠标滚轮来左右移动波形，比 Vivado 方便许多。但用鼠标点击波形信号线，虽然能够查看信号在对应时刻的值，但不能选中信号。

此外，可以点击选中某个信号之后，点击工具栏的"吸边"工具，从而定位到信号发生变化的上一个或下一个时刻，如下图所示。

> [图片描述] Surfer 工具栏的"吸边"工具（定位到信号边沿）按钮位置

点击选中某个信号之后，也可以通过键盘的 `Left` 和 `Right` 方向键来定位信号发生变化的时刻。

点击菜单栏的 "Go to start" 或 "Go to end" 按钮，可快速定位到波形的最开始或最末尾，如下图所示。

> [图片描述] Surfer 菜单栏的 "Go to start" 和 "Go to end" 按钮位置
