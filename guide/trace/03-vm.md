# B. 虚拟机使用指南

## 1. 下载虚拟机

测试环境及相关工具下载地址：

- WSL2 虚拟机镜像：[comp2008_wsl2.tar](https://pan.baidu.com/s/191-6NLBXG3O_hFvtVTKBSw?pwd=srui)
- Ubuntu 虚拟机镜像：[vm.ova](https://otuyernchr.feishu.cn/file/boxcnrClUmwkmCZz5JbPsf7QnCb)
- MobaXTerm 和 VirtualBox：[虚拟机工具.zip](https://otuyernchr.feishu.cn/file/boxcnTjEReejhlrl0DlCte1Lrrf)

如果下载的是 WSL2 虚拟机，则自行搜索如何安装 WSL2 环境，然后导入虚拟机、启动虚拟机即可使用。

如果下载的是 Ubuntu 虚拟机，请参照本页后的"附录"导入虚拟机。

## 2. 导入虚拟机

推荐使用 VirtualBox。下载完毕后，请在 VirtualBox 中导入：管理 -> 导入虚拟电脑，选择对应 OVF 文件。

> [图片描述] VirtualBox 导入虚拟电脑的界面，选择 OVF 文件

## 3. 配置虚拟网卡

VirtualBox 中对于虚拟网卡，需要稍微配置一下才可以使用，具体的配置如下：

> [图片描述] VirtualBox 网络设置界面，配置网卡 1 为 NAT 模式

> [图片描述] VirtualBox 网络设置界面，配置网卡 2 为 Host-Only 模式

> [图片描述] VirtualBox Host-Only 网络的高级设置界面

## 4. 使用 MobaXTerm

为了更好的使用体验，推荐使用 MobaXTerm 通过 SSH 连接到本地的虚拟机进行进一步的操作。压缩包里提供了 MobaXTerm。

在 MobaXTerm 中，点击左上角"会话"。

> [图片描述] MobaXTerm 新建会话界面

在弹出窗口中，点击 "SSH"，按如下内容填写（"远程主机"地址需要自己查询）。填写完毕后，点击"好的"。输入虚拟机密码 `123456`。

> [图片描述] MobaXTerm SSH 会话配置界面，填写远程主机地址、用户名和端口

连接完毕之后，虚拟机可最小化到后台，在命令行中进行操作。左侧可以与虚拟机的文件交互（上传、下载），右侧可以输入命令。

> [图片描述] MobaXTerm 成功连接虚拟机后的操作界面，左侧为 SFTP 文件浏览器，右侧为终端
